# County-Level Economic Factors and High School Graduation Rates in Washington State

This project studies the relationship between county-level economic conditions and high school graduation rates in Washington State. We combine a Washington OSPI graduation dataset with the U.S. Census Bureau SAIPE dataset to analyze income, child poverty, and graduation outcomes.

## Files

- `dataset_setup.py`  
  Cleans and merges the OSPI graduation dataset and the SAIPE socioeconomic dataset. Running this file creates `merged_dataset.csv`.

- `dataset_graph.py`  
  Generates the visualizations used in the report, including static plots and Plotly county maps.

- `dataset_model.py`  
  Runs the predictive models used in the report. This file compares Linear Regression and Random Forest using MSE and R².

- `dataset_test.py`  
  Runs tests on the merged dataset, including row counts, county uniqueness, missingness, and value-range checks.

- `report.pdf`  
  Final report for the project.

## Data Files Needed

Make sure these two data files are in the same folder as the Python files:

- `Report_Card_Graduation_2024-25.csv`
- `est24all.xls`

```bash
pip install pandas seaborn matplotlib plotly scikit-learn openpyxl xlrd
```

## How to Run the Project

Run the following files in this order.

### 1. Create the merged dataset

```bash
python dataset_setup.py
```

This creates:
- `merged_dataset.csv`

### 2. Generate the plots and maps

```bash
python dataset_graph.py
```

This creates:
- `graduation_rate_distribution.png`
- `graduation_rate_by_cohort.png`
- `median_household_income_distribution.png`
- `graduation_rate_vs_median_income.png`
- `graduation_rate_vs_median_income_jointplot.png`
- `child_poverty_distribution.png`
- `graduation_rate_vs_child_poverty.png`
- `graduation_rate_map.html`
- `median_household_income_map.html`

### 3. Run the predictive models

```bash
python dataset_model.py
```

This prints the model results for:
- Linear Regression
- Random Forest Regressor

### 4. Run the tests

```bash
python dataset_test.py
```

## Notes

- All Python files and data files should stay in the same folder.
- Do not rename the input data files unless you also change the filenames in the code.
- The Plotly maps are saved as `.html` files and can be opened in a web browser.
- The figures used in the report are generated by `dataset_graph.py`.
- The model results reported in the paper are generated by `dataset_model.py`.
